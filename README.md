# Trầm Hương Đại Phát

Website bán hàng Trầm Hương Đại Phát được xây dựng theo mô hình fullstack, hỗ trợ hiển thị sản phẩm, quản lý tài khoản, giỏ hàng, đặt hàng và quản trị dữ liệu sản phẩm.

## Công nghệ sử dụng
- Frontend: React, Vite, JavaScript, CSS
- Backend: Node.js, Express
- Database: PostgreSQL
- Công cụ: Git, GitHub, VS Code, pgAdmin 4

## Chức năng chính
- Hiển thị danh sách và chi tiết sản phẩm
- Đăng ký, đăng nhập tài khoản
- Giỏ hàng và đặt hàng
- Quản lý sản phẩm phía admin
- Upload hình ảnh sản phẩm
- Kết nối cơ sở dữ liệu PostgreSQL
- Thanh Toán Bằng MOMO

## Cấu trúc dự án
- `frontend`: giao diện người dùng
- `backend`: API và xử lý dữ liệu
- `database`: file cơ sở dữ liệu
